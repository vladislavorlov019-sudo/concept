Курсовая работа: "Разработка автоматизированной системы отчётности, сбора и визуализации данных"
Репозиторий предназначен для размещения материалов проектного практикума и сопутствующих разделов курсовой работы по теме "Разработка автоматизированной системы отчётности, сбора и визуализации данных для компании-производителя сыров".
Описание системы
Система предназначена для автоматизации сбора данных из разрозненных источников (1С, CRM Pro Data, ручные файлы Excel) и их консолидации в единой облачной базе данных Clickhouse. С помощью инструментов автоматизации (Power Automate или n8n) реализуются ETL-процессы, обеспечивающие актуальность данных для четырех целевых групп пользователей.
Основные группы отчётности:
1. Высшее руководство — стратегические показатели и общая динамика производства.
2. Начальники отделов — данные для принятия оперативных решений.
3. Рядовые сотрудники — аналитическая информация для повседневной работы.
4. KPI — мониторинг эффективности и автоматизированный расчёт заработной платы.

Структура проекта
В репозитории представлены следующие этапы проектирования:

**Раздел A: Концептуальное проектирование**
- A1) Эссе.md — Обоснование актуальности системы для сырного производства.
- A2) Суммаризация.md — Краткое изложение функциональных требований и архитектуры.
- A3) Карта знаний — Визуализация связей между источниками данных и потребителями отчётов.

**Раздел B: Варианты реализации**
- B1) Концепт №1 — Реализация на стеке Microsoft (Power Automate, Power BI).
- B2) Концепт №2 — Альтернативная реализация (n8n, Clickhouse, DataLens).

**Раздел C: Архитектура системы (C4 Model)**
- C1) Контекстная диаграмма — Взаимодействие системы с внешними сущностями (1С, CRM).
- C2) Контейнерная диаграмма — Описание контейнеров сбора (n8n), хранения (Clickhouse) и визуализации.
- C3) Диаграмма компонентов — Внутренняя логика модулей расчёта KPI и разграничения прав доступа.
- C4) Диаграмма кода — Детализация алгоритмов обработки данных.

**Раздел D: Пользовательский опыт**
- D1) Диаграмма клиентских путей — Процесс взаимодействия пользователя с отчётностью от ввода данных до получения результата.
